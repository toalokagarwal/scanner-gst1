# Scanner GST Ready Reckoner

India's GST Operating System — Updated for GST 2.0 (Post 56th Council, Sep 2025)

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

## Stack

- React 18 + Vite
- Single-file app (`src/App.jsx`)
- Zero external UI dependencies
- Vercel-ready (`vercel.json` included)
